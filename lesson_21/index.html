Задача №1 Яка назва змінної вірна?

  1. let user-name
  2. let UserName
  3. let userName
Відповідь Варіант 3 - це є правильна відповідь. У JS для написання змінних використовується стиль lowerCamelCase.

Задача №2 Чи буде помилка при виконанні наступної програми?

  1. const userName = 20
  2. userName = "Іштван"
Відповідь Так, данний варіант є помилкою, бо значення константи неможливо змінити.

Задача №3 Що потрапить в консоль?

  1. let numOne = 20
  2. let numTwo = "20"
  3. let numSumm = numOne + numTwo
  4. console.log(numSumm)
Відповідь В консоль потрапить рядок "2020", оскільки бінарний оператор плюс(+) не змінює тип данних.
